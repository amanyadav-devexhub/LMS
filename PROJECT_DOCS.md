# LeaveOS — Comprehensive Project Documentation

## Project Overview
LeaveOS is a modern, premium-grade Leave Management System (LMS) built with Django. It provides a robust platform for managing employee leaves, department hierarchies, and detailed user profiles with a focus on high-end aesthetics and user experience.

---

## 1. Core Modules

### 👤 User & Profile Management (`users` app)
- **Authentication**: Secure Login, Logout, and Registration systems.
- **Role-Based Access Control (RBAC)**: Fine-grained permissions for different organizational roles (Admin, HR, Manager, Team Lead, Employee).
- **Ultra-Premium Profiles**: A multi-tabbed profile interface featuring:
  - **Basic Details**: Contact info, personal details, and profile photo.
  - **Salary Details**: Protected section for compensation info (HR/Admin/Manager only).
  - **Bank Details**: Secure storage for financial info.
  - **Verification**: Aadhaar and PAN tracking.
  - **Additional Information**: Blood group, notes, and medical info.
- **Profile Photo (Avatar)**: Interactive photo upload with instant preview.
- **Department & Designation Management**: Centralized management of organizational structure.

### 📅 Leave Management (`leaves` app)
- **Self-Service Leave Application**: Employees can apply for leaves with automated balance checking.
- **Multi-Level Approval Workflow**: Managers and HR can approve or reject leave requests.
- **Leave Analytics**: Visual reporting and data insights for HR (Active today, New joiners, etc.).
- **Leave History**: Comprehensive logs of past, present, and pending leave requests.

## 2. Dashboard Ecosystem
LeaveOS features specialized dashboards tailored to each user role, providing relevant tools and metrics at a glance.

- **Employee Dashboard**: Focus on applying for leaves and viewing personal balances.
- **Team Lead (TL) Dashboard**: Team-centric view for managing direct reports.
- **Manager Dashboard**: *Ultra Premium UI* with tabbed management for team queues, history, and profile.
- **HR Dashboard**: Organizational oversight, analytics, and employee management.
- **Admin Dashboard**: Total system control including role/permission assignments.
